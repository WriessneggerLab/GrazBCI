# GrazBCI

Presentation paradigm for GrazBCI MI/ME [1], implemented in Psychopy.

## Default Timings

Intro: 5s
Trial: 
  - Fixation cross: 3.00s
  - Pre-cue: 1.50s
  - Cue: 5.00s
  - Blank: 2.00s
End: 5s
